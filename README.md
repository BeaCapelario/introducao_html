Este repositório contém os exercícios e projetos iniciais focados no aprendizado de **HTML5** e **CSS3**. É o ponto de partida da minha jornada no desenvolvimento web, onde explorei a estrutura fundamental de páginas e a semântica da web.

---

###  O que foi explorado:
* **Semântica:** Uso correto de tags como `<header>`, `<main>`, `<footer>` e `<section>`.
* **Formulários:** Criação de campos de entrada, validações simples e botões.
* **Mídia:** Incorporação de imagens, vídeos e links externos.
* **Estilização Básica:** Introdução ao CSS para dar vida ao esqueleto HTML.

---

| Tecnologia | Descrição |
